# API 文档模板

## 接口信息

| 属性 | 值 |
|------|-----|
| **接口名称** | [接口名称] |
| **接口描述** | [功能描述] |
| **请求方法** | GET / POST / PUT / DELETE |
| **请求路径** | `/api/v1/resource` |
| **Content-Type** | application/json |
| **认证方式** | Bearer Token |

---

## 请求参数

### Path Parameters

| 参数名 | 类型 | 必填 | 描述 | 示例 |
|--------|------|------|------|------|
| id | Long | 是 | 资源ID | 1 |

### Query Parameters

| 参数名 | 类型 | 必填 | 默认值 | 描述 | 示例 |
|--------|------|------|--------|------|------|
| page | Integer | 否 | 1 | 页码 | 1 |
| size | Integer | 否 | 20 | 每页数量 | 20 |

### Request Body

| 参数名 | 类型 | 必填 | 约束 | 描述 |
|--------|------|------|------|------|
| name | String | 是 | 长度 1-100 | 名称 |
| status | Integer | 是 | 枚举: 0,1,2 | 状态 |

---

## 响应参数

### 成功响应 (200)

| 参数名 | 类型 | 描述 |
|--------|------|------|
| code | Integer | 响应码，0 表示成功 |
| message | String | 响应消息 |
| data | Object | 响应数据 |

### Data 对象结构

| 参数名 | 类型 | 描述 |
|--------|------|------|
| id | Long | 资源ID |
| name | String | 名称 |
| createdAt | String | 创建时间 (ISO 8601) |

---

## 示例

### Request Sample

```json
{
    "name": "示例名称",
    "status": 1
}
```

### Response Sample

```json
{
    "code": 0,
    "message": "success",
    "data": {
        "id": 1,
        "name": "示例名称",
        "createdAt": "2024-01-01T10:00:00Z"
    }
}
```

---

## 错误码

| 错误码 | HTTP Status | 描述 | 解决方案 |
|--------|-------------|------|----------|
| 10001 | 400 | 参数校验失败 | 检查请求参数格式 |
| 10002 | 404 | 资源不存在 | 确认资源ID正确 |
| 10003 | 500 | 服务内部错误 | 联系管理员 |
